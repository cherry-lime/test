# Test Advisor
Before running, make sure a database URL is included in the .env file.

### Install Prisma and generate Prisma client:
```
$ npm install prisma --save-dev
$ npx prisma generate
```

### Install the node modules:
```
$ npm install
```

### Run the web app:
```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
